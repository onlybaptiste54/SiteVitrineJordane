1. Présentation du projet

MERINO est une agence immobilière indépendante basée dans le Grand Nancy. Le projet

3. Architecture du tunnel  
 
Le tunnel est composé de trois pages enchaînées :  
 
Page  Description  
Page 1 — Landing page  Page d'entrée commune. Présentation de l'offre, formulaire 
d'estimation. Identique pour tous les visiteurs.  
Page 2 — Chargement  Page intermédiaire affichant une barre de progression pendant 15 
à 18 secondes. Introduit l'agent qui va rappeler le visiteur.  
Page 3 — Confirmation  Affiche la fourchette d'estimation, la fiche de l'agent, les boutons 
de contact, et le guide téléchargeabl e. 
 
3.1 Logique de routage par agent  
La landing page (Page 1) est commune à tous les visiteurs. En revanche, les pages 2 et 3 
affichent des informations différentes selon l'agent à l'origine de la publicité qui a amené le 
visiteur.  
 
Le routage se fait via  un paramètre dans l'URL transmis depuis la publicité Meta :  
– Publicité de Jordane → URL contient ?agent=jordane  
– Publicité de Louis → URL contient ?agent=louis  
 
Ce paramètre est lu côté client (JavaScript) et détermine quel agent afficher sur les pages 2 et 3 : 
photo, prénom, numéro de téléphone, message personnalisé, et vidéo.  
Le paramètre agent doit être propagé d'une page à l'autre tout au long du tunnel, y com pris lors 
de la redirection vers la page de confirmation.

7. Moteur d'estimation  
 
7.1 Principe  
Le moteur d'estimation est un calcul côté client (JavaScript) qui génère une fourchette de prix 
indicative à partir des données saisies dans le formulaire. Il ne se connecte à aucune API 
externe.  
 
La fo urchette est calculée à partir d'un prix au m² de référence par type de bien, ajusté par des 
coefficients selon l'état du bien et le secteur géographique.  
 
7.2 Données d'entrée  
– Type de bien (appartement, maison, terrain…)  
– Surface en m²  
– Adresse (utilisée po ur affichage uniquement à ce stade — pas de géolocalisation 
automatique)  
 
7.3 Logique de calcul  
Prix de base = surface × prix_m2_reference[type_bien]  
Prix bas = prix de base × 0,92  
Prix haut = prix de base × 1,08  
 
Valeurs de référence indicatives pour le G rand Nancy (à affiner par MERINO) :  
 
Type de bien  Prix au m² de référence  
Appartement  2 200 €/m²  
Maison  2 600 €/m²  
Terrain  120 €/m²  
Local commercial  1 800 €/m²  
 
Ces valeurs sont des paramètres modifiables facilement dans le code (objet de configuration). 
MERINO pourra les ajuster sans intervention du développeur si le code est correctement 
structuré.

10. Spécifications techniques  
 
10.1 Performance mobile  
70% du trafic Meta arrive sur mobile. Les performances mobiles sont non négociables.  
– Temps de chargement < 3 secondes sur 4G (score Google PageSpeed > 80)  
– Design mobile -first 
– Images compressées en WebP  
– Formulaire visible sans scroller sur mobile  
 
10.2 Stack technique  
– Pages statiques ou framework léger (Next.js, Nuxt, HTML/CSS/JS vanilla)  
– Pas de WordPress  
– HTTP S obligatoire (requis par Meta Ads)  
– Hébergement et nom de domaine gérés par le développeur  
 
10.3 Compatibilité navigateurs  
– Chrome, Firefox, Safari — desktop et mobile  
– iOS Safari 14+ et Android Chrome  
 
10.4 URLs et structure  
– Landing page : domaine principal ou sous -domaine dédié (ex : vendre.merino -immo.fr)  
– Page de chargement : /chargement  
– Page de confirmation : /confirmation  
– Les trois pages sont isolées — aucun lien vers le site vitrine, pas de menu  
 
11. Mentions légales et R GPD

– Mention RGPD sous le formulaire : "Vos données ne seront jamais transmises à des tiers."  
– Conformité RGPD pour la collecte de données personnelles (prénom, téléphone)
